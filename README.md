# Yuzu / Nintendo Switch - Emulator and Console Tips

## Setup
* *yuzu_install.exe*
* Copy *ProdKeys.zip* and extract *prod.keys* and *title.keys* into [C:/Users/%USERNAME%/AppData/Roaming/yuzu/keys](C:/Users/%USERNAME%/AppData/Roaming/yuzu/keys)
